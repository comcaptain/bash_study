# How to create a Gradle build?

Very simple, just run `gradle init` . If you use default option for all choices, then you'll get an empty build like below:

```
├── build.gradle  
├── gradle
│   └── wrapper
│       ├── gradle-wrapper.jar  
│       └── gradle-wrapper.properties  
├── gradlew  
├── gradlew.bat  
└── settings.gradle  
```
